This plugin useful modules for Live Composer, like ACF Placeholder.

== Description ==

At the moment we add only single module, which allows you to print Advanced Custom Fields and style them in Live Composer way.

This plugin required Live Composer to be installed, because it adds modules for it. If want to offer any features please contact us
via our [contact-us](https://hosting.review/contact-us/) form.

= Good to know =

- This plugin doesn't have any specific api of its own.
- Plugin uses Live Composer method of adding into it new modules everything else is pure Wordpress API.

== Installation ==

1. Download and upload plugin or find it in Wordpress plugins directory and install.
2. Make sure you have Live Composer 3+ and Advanced Custom Fields or Advanced Custom Fields Pro plugins.
3. Create fields add them to preferred post types, fill them in.
4. Add ACF Placeholder module to location you want ACF Field to be displayed.
